Project idea

My project of choice is building a smart mirror which function as a mirror and display show the daily news. In this project, we have to build a mirror which have a display attach behind it. To make it run smartly, we need to make use of the minicomputer like Raspberry Pi or Arduino. What is left now is programing the system so that it can display the time, daily news, weather of the day or the maps for our destination. Furthermore, we can play around with the idea and adding some optional features like touch screen, smart AI with voice control (Alexa, Google assistance or Siri) and multiple profiles for each family member so they can custom it by their own taste.

It took averagely 30 minutes in the morning for a person to get ready for the day, and then to get the earliest news it took 30 minutes more, not only that we also have to do those activities in other time of the day. Imagine that we can emerge two of that together and we will save a lot of our time. By saving those time, we will not need to be rush in the morning commute anymore. For example, if everyone in the city own this mirror, the city life will be much slower than at the present.

Features:
  • Firstly, we can update with the daily information during our personal hygiene. We will find out everyday weather, and that with be really helpful for planning the day. If it is raining outside or will be rain, we would know that umbrella is needed. In addition, we can set it up to display the daily traffic news so that we may be informed that which road is crowed or even having accident. Speaking from my experience, I always forget to look at the news because of the lack of morning prepare time, and there are a few times it was showered by I did not bring any umbrella or raincoat, so I was soaking. By that, preparing for the day will be more practical and effective, we will have less time wasted on the crowed road or taking cover on the rainy day.
  • The second feature is separated profile for each people. This mean that every user can customize the mirror to display different thing to fit their own taste. For example, dad want to get updated with sport news but mom prefer gossip, now there is different user function we can look at different thing. This feature may look familiar because we already use it in our personal computer. However, in computer every time we want to use as different users we need to log in, so for this device we can use the face id to scan for the user and it will log out after we leave the mirror for a minute
  • Thirdly, there will be an advices and tips feature to help the users when they need. For example, if that day the owners want to find a calm and relaxing place, the mirror will suggest some places. With the AI assistant, such as Alexa, Google Assistant or Siri, asking questions is much more helpful. There will be more useful tips if we know a good question to ask. In addition, it can connect on with our smart house system, so it can give us about some house keeping advices. Everyone wants to keep their house as clean as possible but do not know where to begin, so the mirror may have some advice for that too.
  • The last attractive feature is just for fun, it is the emotion tracker function. It will scan our facial structure to find out our emotion so that it will know if we are happy or not. After knowing our emotion, the AI will reply with the appropriate response. For example, if we come back with a sad face, it will show us cat video to cheer us up or if we are happy it will congratulate us to make us even happier.
